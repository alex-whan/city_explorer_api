## Getting Started

User must run the following library installs in order to build and run this application:

* `npm i -S express`
* `npm i -S dotenv`
* `npm i -S cors`
* `npm i -S superagent`
* `npm i -S node` - user must also have Node.js installed

* `npm init` followed by:

          npm install express dotenv cors superagent
